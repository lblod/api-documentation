Defined in node_modules/@lblod/marawa/rdfa-helpers.js:10

Enriches a rich node with semantic properties:

  • rdfaPrefixes: map of prefixes at the current node
  • rdfaAttributes: resolved (non-prefixed) RDFa attributes set on the node
  • rdfaContext: array of rdfaAttributes from the top to the current node

Parameters:

  • richNode RichNode

    Rich node to expand the RDFa from

  • parentContext Array

    RDFa context (array of rdfaAttributes) of the node's parent

  • parentPrefixes Object

    RDFa prefixes defined at the node's parent level
